* v: [a] [s] [d] [f] [g] [h] [j] [k] [l] [ñ] [ḉ]
* sub: [z] [x] [c] [v] [b] [n] [m] [,] [.] [-]
* pitchbend: [l_shift + 1] [l_ctrl - 1]
